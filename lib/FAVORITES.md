### NPM

```bash
# initialize npm package
$ npm init -y

# install package
$ npm install <package>

# install and save package as dependency
$ npm install --save <package>
$ npm i -S <package>

# install and save package as dev dependency
$ npm install --save-dev <package>
$ npm i -D <package>
```
